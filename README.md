# Elga Cloud – WHMCS Domain Reseller Module

Elga Cloud Domain Reseller Module is a WHMCS integration that allows hosting providers and resellers to sell and manage domain registrations directly from WHMCS using Elga Cloud API.

This module supports PHP 7+ and modern WHMCS versions, providing automated domain provisioning, renewal, and management.

---

## 🚀 Features

- Domain Registration  
- Domain Renewal  
- Nameserver Management  
- Client Area Domain Management  
- Admin Configuration Panel  
- Auto Domain Sync  
- Secure API Authentication  
- Lightweight & Fast  
- Supports Multiple TLDs  
- Clean Code Structure  
- Shared Hosting & VPS Compatible  

---

## ✅ Requirements

- PHP 7.0 or higher (PHP 8.x supported)
- WHMCS 8.x+
- MySQL 5.6+
- cURL Enabled
- Allow_url_fopen Enabled
- ionCube Loader (optional)
